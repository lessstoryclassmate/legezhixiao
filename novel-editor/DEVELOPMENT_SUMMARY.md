## 配置信息

### 🗄️ 数据库配置
- **系统数据库**: 172.16.16.3:3306/novel_data
- **用户数据库**: 172.16.16.2:3306/novel_user_data
- **数据库类型**: MySQL 8.0

### 🔑 API配置  
- **SiliconFlow API**: 已配置密钥
- **支持模型**: deepseek-v3, 千问
- **功能**: 内容生成、分析、优化、建议

### 🌐 服务端口
- **后端API**: 8000
- **前端界面**: 3000 (开发) / 80 (生产)
- **允许端口**: 22, 80, 8000, 8080, 30080

## 核心功能特点

### 🎯 AI功能亮点
1. **智能内容生成**: 基于上下文的小说内容创作
2. **深度文本分析**: 情感曲线、节奏分析、人物关系
3. **剧情建议系统**: 智能推荐情节发展方向
4. **文本优化**: 自动优化文笔和表达

### 📝 编辑器特色
1. **VSCode风格**: 熟悉的深色主题界面
2. **实时协作**: 支持多人同时编辑
3. **章节管理**: 直观的章节组织和导航
4. **自动保存**: 防止内容丢失

### 🏗️ 架构优势
1. **微服务架构**: 前后端分离，易于扩展
2. **容器化部署**: Docker + K8S，便于运维
3. **双数据库**: 系统与用户数据分离
4. **高可用性**: 负载均衡和故障转移
