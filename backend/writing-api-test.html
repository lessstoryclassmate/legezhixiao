写作工具 API 测试面板

测试新创建的写作工具后端API端点功能

📚 项目管理 API

GET
/api/writing/projects

获取用户的所有项目

POST
/api/writing/projects

创建新项目

POST
/api/writing/projects/template

生成项目模板

📝 章节管理 API

GET
/api/writing/projects/{projectId}/chapters

获取项目章节列表（需要先创建项目）

👥 角色管理 API

GET
/api/writing/projects/{projectId}/characters

获取项目角色列表

🌍 世界观设定 API

GET
/api/writing/projects/{projectId}/worldbuilding

获取项目世界观设定

📊 写作统计 API

GET
/api/stats/projects/{projectId}/stats/overview

获取写作概览统计

GET
/api/stats/projects/{projectId}/stats/heatmap

获取写作热力图数据

🔍 系统状态

GET
/health

检查系统健康状态
